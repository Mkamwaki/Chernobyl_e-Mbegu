eMbegu

Chernobyl Hackathon 2023

Description

e-Mbegu is a mobile application that will help farmers distinguish geniune
seeds from fake seeds when purchasing them from dealers. The farmer will enter
a unique code from the packet storing the seeds and once the code is entered the
app will determine whether the seeeds are geniune or not.

Endpoints

The following are the endpoints that are available in the API

  • GET / - This documentation page
  • POST /login - login with phone and password
  • POST /register - register with phone, fullname, ward, region and password
  • GET /regions - returns a list of regions in Tanzania
  • GET /wards?region={region_name} - returns all wards given a region
  • GET /verify?number={seed_number} - checks if the seed is original or fake given the packet number
